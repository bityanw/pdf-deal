# 更新日志

## v19 - 2026-03-20

### 新增功能

#### 费用明细表功能
在PDF发票合并工具中新增费用明细表功能，支持：

1. **自动解析发票信息**
   - 从上传的PDF发票中自动提取关键信息
   - 支持提取：开票日期、发票号码、发票类型、金额等

2. **费用明细表预览**
   - 点击"预览费用明细表"按钮查看表格数据
   - 表格包含以下字段：
     - 日期：从发票中提取的开票日期或乘车日期
     - 项目名称：留空（用户可后续填写）
     - 类别：留空（用户可后续填写）
     - 金额：发票的价税合计或总金额
     - 其他：发票号 + 发票类型
     - 小计：当前行的总金额
   - 自动计算合计行

3. **导出Excel功能**
   - 点击"导出费用明细表"按钮导出Excel文件
   - 文件名格式：`业务费用明细表_YYYY-MM-DD.xlsx`
   - Excel格式符合业务费用明细表模板规范
   - 包含边框、样式和合计行

### 使用方法

1. 上传PDF发票文件（支持最多999份）
2. 选择合并方式和展示内容
3. 点击"免费转换"按钮
4. 转换成功后：
   - 点击"下载合并PDF"下载合并后的PDF文件
   - 点击"预览费用明细表"查看费用明细数据
   - 点击"导出费用明细表"导出Excel格式的费用明细表

### 技术实现

- 新增类型定义：`ExpenseDetailRow`
- 新增工具函数：
  - `convertToExpenseDetailRows()`: 将发票数据转换为费用明细表行数据
  - `generateExpenseDetailExcel()`: 生成Excel格式的费用明细表
- 修改组件：`InvoiceMergeTool.tsx`
  - 集成发票解析功能
  - 添加费用明细表预览UI
  - 添加Excel导出功能

### 数据映射规则

| Excel列 | 数据来源 | 说明 |
|---------|---------|------|
| 日期 | `invoiceDate` 或 `departureDate` | 优先使用开票日期，其次使用乘车日期 |
| 项目名称 | - | 留空，用户后续填写 |
| 类别 | - | 留空，用户后续填写 |
| 金额 | `amountWithTax` 或 `totalAmount` | 价税合计或总金额 |
| 其他 | `invoiceNumber` + `invoiceType` | 发票号码和发票类型组合 |
| 小计 | 计算值 | 等于金额（当前实现） |
